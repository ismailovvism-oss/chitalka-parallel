# Changelog — Параллельная читалка

Хронология для самопогружения в контекст. Детали формата — `SPEC.md`, рабочий
контекст — `CLAUDE.md`. Новое сверху.

## 2026-06-18 — сворачиваемые callout’ы (sw `v18`)

### `parser.js` / `style.css` — фолд callout’ов как в Obsidian
- `renderBlocks`: `> [!тип]+` → раскрытый сворачиваемый callout, `> [!тип]-` →
  свёрнутый; без знака — статичный `<div>` как раньше. Регэксп `[!тип]([+-]?)`.
  Сворачиваемые рендерятся как `<details class="callout-foldable">` + `<summary>`.
- CSS: маркер-стрелка у `<summary>`, тело свёрнутого скрыто явным правилом
  `details.callout-foldable:not([open]) > .callout-body{display:none}` (не полагаемся
  на UA-стиль `<details>`). Тот же словарь и рендер — в вычитке (редактор).

## 2026-06-16 — общий словарь с вычиткой, заметки, недопереведённые секторы

Вычитка (`translator-panel`) стала редактором этого формата «на месте»; читалка
доработана под общий контракт.

### `parser.js` — рендер форматирования (sw `v17`)
- `inlineMd`: подчёркивание `<u>…</u>`.
- `renderGroup`→`renderBlocks`: цитата `> ` → `<blockquote>`; списки `- `/`1. ` →
  `<ul>/<ol>`; callout `> [!тип] заголовок` → `.callout.callout-<тип>` (тип
  произвольный, дефолтная подпись из `CALLOUT_LABELS`, неизвестный тип — базовый стиль).
- `parseFile`: регионы `<!-- note --> … <!-- /note -->` пропускаются (личные
  редакторские заметки автора не публикуются). Парсер остаётся DOM-free.

### `app.js` / `style.css` — недопереведённые секторы
- `renderChapter`: текст-пара с одним показанным языком → `.solo` (во всю ширину);
  оригинал без перевода (в двуязычной книге) → `.awaiting` (пунктирная акцентная
  грань + значок «⟳», слово — в `title`, видно только при параллельном чтении).
- CSS форматирования (blockquote/списки/callout/`<u>`) — тема-зависимый.

### Зачем
Гибридные книги (моноязычный «хребет» + вставные параллельные цитаты) и публикация
недопереведённых книг: сектор только в оригинале = «перевод готовится», сектор только
в переводе = основной текст во всю ширину, сектор в обоих с одним id = параллель.
