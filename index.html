      Настройки

      Справка

      Жесты

      • Тап по абзацу — когда показан один язык, временно раскрывает второй (повторный тап прячет).
      • Свайп влево/вправо — следующая / предыдущая глава.

      Кнопки

      • 📚 — все книги (библиотека).
      • ☰ — оглавление и список закладок.
      • 🔍 — поиск по книге (оба языка; арабский — без учёта огласовок).
      • 🔖 — поставить/снять закладку на текущем месте.
      • ⇅ — видимость языков: оба → только один → только другой.
      • ⚙ — настройки.
      • ‹ › — предыдущая / следующая глава.
      • стр. N — переход к номеру страницы; скан — фото страницы (если есть).

      Клавиатура

      • ← → — листать главы. Esc — закрыть окно.

      Настройки

      • Тема — светлая, сепия, тёмная или «как в системе».
      • Раскладка — вертикально, в две колонки или по ориентации экрана.
      • Шрифт, размер, интервал — отдельно для каждого языка; плюс поля страницы.
      • Две колонки — ширина колонок и «RTL-язык справа».
      • Порядок пары, выравнивание, режим сносок.
      • Экспорт/Импорт — резервная копия настроек и закладок в файл.

      Закладки

      • Список и заметки — в оглавлении (☰). «↗» копирует ссылку на это место — ею можно поделиться.

      Офлайн

      • «Установить на экран» из меню браузера — читалка работает без сети, прочитанное сохраняется в кеш.
