# Бэклог

Идём по порядку. Отмечаем сделанное `[x]`.

## Сильно меняют опыт
- [x] **1. Запоминание позиции + «Продолжить чтение».** Хранить активный сектор (а не только главу), возвращаться ровно туда. В библиотеке — строка «Продолжить: <книга>, стр. N».
- [x] **2. Офлайн / PWA.** Service worker + манифест: установка на домашний экран, работа без сети, кеш контента (в т.ч. внешних книг с гитхаба).
- [x] **3. Поиск по книге.** Полнотекстовый по главам, переход к сектору. Для арабского — нормализация (огласовки/хамза).

## Учебные, под билингву
- [x] **4. «Подсмотреть» перевод по тапу.** Когда показан один язык, тап по абзацу временно раскрывает второй.
- [x] **5. Закладки и заметки/выделения** на сектор; быстрый список и переход. Deep-link на сектор (`?book=…&s=…`).

## Типографика
- [x] **6. Межстрочный интервал и поля** — слайдеры, как у размера шрифта.
- [x] **7. Соотношение колонок** в режиме «две колонки»; RTL-осознанный порядок (арабский справа).

## Мелочи
- [x] **8. Клавиатура** — стрелки (главы/страницы), Esc (закрыть оверлей).
- [x] **9. Прогресс** — полоска/процент по главе и книге.
- [x] **10. Тема «сепия».**

# Блок v4 — идеи (обсуждение 11.06.2026)

## Педагогические (изучение языка)
- [x] **Режим самопроверки** — 4-е состояние кнопки ⇅ (двуязычные книги): виден оригинал, перевод свёрнут в заглушку «· · · перевод · · ·», тап по паре раскрывает/прячет. Активное припоминание — читаешь оригинал, вспоминаешь смысл, проверяешь.
- [ ] **Экспорт выделений в Anki/CSV** — выделения уже копятся; отдать файлом «выделенная фраза + текст парного сектора на втором языке». Чтение само наполняет колоду повторения.
- [ ] **Словарь по тапу, этап 1 (без бэкенда)** — выделенное слово → ссылка наружу (Almaany / Wiktionary) в новой вкладке. Полноценный офлайн-словарь — отдельная большая задача (источник так и не выбран).
- [ ] **Разметка «голосов»** (матн / хадис / комментарий разным стилем) — перенесено из SPEC v2; для религиозных текстов сильно поднимает читабельность.

## Удобство чтения
- [ ] **Иллюстрации в тексте** — поддержать `![подпись](img/...)` внутри сектора (parser.js: inlineMd) + тап для увеличения. Сейчас есть только сканы целых страниц, картинок внутри текста парсер не знает.
- [ ] **Озвучка сектора** (Web Speech API, без зависимостей) — кнопка «прослушать абзац» у активной пары, оба языка. Качество арабского зависит от системных голосов — принять как есть.
- [ ] **Печать / PDF главы** — print-стилизация: пары в две колонки, сноски в конце. «Бумажная билингва» бесплатно.

## Конвейер контента
- [ ] **Импорт существующих книг**: формат Вычитки как промежуточный — Клод выравнивает оригинал↔перевод по секторам, автор выборочно проверяет в Вычитке, конвертер (import-tam.js как образец) переносит в читалку. Один импортёр на исходный формат.
